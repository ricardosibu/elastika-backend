
# Elastika Test Backend

API que permite servir a una app de chat con inteligencia artificial.




## Tech Stack


**Server:** Node, Express
**Database:** MongoDB (Usando MongoDB Atlas)



## Installation

Install my-project with npm

```bash
  npm install backend
  cd backend
  npm run dev
```
